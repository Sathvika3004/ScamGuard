
# ScamGuard

ScamGuard is a terminal-style web application that detects scam messages in real time.

## Features
- Paste-and-scan message analysis
- Risk classification: Safe, Suspicious, Scam
- Terminal-style UI for clarity
- Rule-based detection logic (MVP)

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js (logic)

## Future Scope
- AI-based detection using Google Vertex AI
- Firebase integration
- Browser extension and mobile app
